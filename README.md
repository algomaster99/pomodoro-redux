# Pomodoro Clock
React-Redux implementation of Pomodoro Clock

## Build & Run
1. Clone the repository.

   ```
   $ git clone https://github.com/algomaster99/pomodoro-redux
  
   $ cd pomodoro-redux
   ```
  
2. Install all dependencies

   `$ npm install`

3. Run the app on localhost:3000/
  
   `$ npm start`
